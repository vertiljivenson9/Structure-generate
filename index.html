Política de Privacidad

Esta Política de Privacidad describe cómo Generator Structure recopila, utiliza y protege tu información cuando utilizas nuestra aplicación web.

Recopilación de Información

Generator Structure está diseñada para funcionar completamente en tu navegador. No recopilamos, almacenamos ni transmitimos ninguna información personal:

  • No almacenamos datos en servidores externos
  • Todo el procesamiento ocurre localmente en tu dispositivo
  • No utilizamos cookies de seguimiento
  • No compartimos información con terceros

Archivos JSON

Los archivos JSON que subes para generar estructuras de proyectos se procesan exclusivamente en tu navegador. No se envían a ningún servidor y se eliminan de la memoria cuando cierras la pestaña.

Seguridad

Implementamos las siguientes medidas de seguridad:

  • Todo el procesamiento ocurre localmente
  • No se almacenan datos sensibles
  • Validación de entrada para prevenir errores
  • Protección contra scripts maliciosos

Cambios en esta Política

Podemos actualizar esta Política de Privacidad ocasionalmente. Te notificaremos sobre cambios significativos publicando la nueva política en esta página.

Términos de Servicio

Bienvenido a Generator Structure. Al utilizar nuestra aplicación web, aceptas los siguientes términos y condiciones.

Uso del Servicio

Generator Structure es una herramienta web que te permite crear estructuras de proyectos a partir de archivos JSON. Puedes utilizar el servicio de forma gratuita para:

  • Generar estructuras de proyectos personales
  • Crear plantillas para proyectos de equipo
  • Organizar la arquitectura de tus aplicaciones
  • Experimentar con diferentes estructuras de archivos

Responsabilidades del Usuario

Como usuario, te comprometes a:

  • Utilizar el servicio de manera legal y ética
  • No intentar acceder a áreas no autorizadas del sistema
  • No utilizar el servicio para actividades maliciosas
  • Verificar que los archivos JSON no contengan código malicioso

Limitación de Responsabilidad

Generator Structure se proporciona "tal cual" sin garantías de ningún tipo. No nos hacemos responsables por:

  • Pérdida de datos o archivos

  • Problemas derivados del uso de archivos generados

Propiedad Intelectual

Generator Structure y su contenido son propiedad de sus desarrolladores. Puedes utilizar la aplicación libremente, pero no puedes:

  • Copiar o distribuir el código fuente sin permiso
  • Utilizar la marca "Generator Structure" sin autorización
  • Revender el servicio como propio

Terminación
